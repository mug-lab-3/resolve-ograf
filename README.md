# resolve-ograf

DaVinci Resolve の OGraf (Open Graphics) 機能を用いたグラフィックスのサンプル集です。

## 免責事項

このリポジトリに含まれるコードは、OGraf の仕様を理解するためのサンプル、または実験目的で作成されたものです。
**実用性は考慮されておらず、プロダクション環境での使用には適していません。** 
あくまで学習用のリファレンスとしてご利用ください。

## 含まれているサンプル

- **YouTubePolaroid**: Web API を用いた外部データ取得とアニメーションのサンプル。
- **WebPageTitle**: Web ページのタイトルのようなアニメーション。
- **EventMonitor**: OGraf のライフサイクルイベントをモニターするためのデバッグツール。
- **その他**: `resolve/Examples` ディレクトリにいくつかの小規模なサンプルが含まれています。

## OGraf について

OGraf (Open Graphics) は、DaVinci Resolve で HTML/JavaScript を使用してカスタムグラフィックスを作成するためのフレームワークです。
詳細な仕様については、`resolve/Documentation` 内のドキュメント（英語）を参照してください。
