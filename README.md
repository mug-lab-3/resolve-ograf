# resolve-ograf

DaVinci Resolve の OGraf (Open Graphics) 機能を用いたグラフィックスのサンプル集です。

## 免責事項

このリポジトリに含まれるコードは、OGraf の仕様を理解するためのサンプル、または実験目的で作成されたものです。
**実用性は考慮されておらず、プロダクション環境での使用には適していません。** 
あくまで学習用のリファレンスとしてご利用ください。

## 含まれているサンプル

- **YouTubePolaroid**: Web API を用いた外部データ取得とアニメーションのサンプル
- **WebPageTitle**: Webページ風
- **EventMonitor**: OGraf のイベントをモニターするためのデバッグツール
